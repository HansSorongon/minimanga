## Features

-   Searching for manga
-   Getting manga chapters
-   Manga descriptions (wip ;-;)
-   Dark mode! Read before you go to bed!
-   Reading manga! <br/>

For now minimanga only supports online mode.

### Built with

-   Native Base
-   React Native
-   Axios (Mangadex API)
-   Expo
